# really_go_orm
锐力golang数据库辅助模块
2022年10月20日发布第一版本
## 1. 项目介绍
本项目是一个golang数据库辅助模块，主要用于简化数据库操作，提高开发效率。
## 2. 项目特点
- 1. 支持多数据库
1. 支持mysql
2. 支持mssql
3. 支持sqlite
4. 支持oracle
5. 支持postgresql
6. 支持db2
7. 支持sqlserver
8. 支持tidb
9. 支持clickhouse
10. 支持firebird
11. 支持cassandra
12. 支持hbase
13. 支持redis
14. 支持mongodb
15. 支持elasticsearch
16. 支持memcache
17. 支持kafka
18. 支持rabbitmq
 
- 2. 支持多种数据库操作
1. 支持数据库连接
2. 支持查询
3. 支持插入
4. 支持更新
5. 支持删除
6. 支持事务
7. 支持存储过程
8. 支持自定义函数
9. 支持自定义函数
